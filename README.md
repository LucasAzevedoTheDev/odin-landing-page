# Landing Page Project - The Odin Project

This project is a test and consolidation of my basics skills in HTML and CSS acquired throughout The Odin Project Foundations course. It's a landing page built from scratch to practice layout structure, styling, and general web development concepts.

## Features practiced 

- HTMl best practices 
- Section organization
- Color palettes and typography
- Header (with navigation links) and footer 
- Basic CSS properties 
- Extensive use of Flexbox
